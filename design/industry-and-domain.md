# 行业与领域选择

游戏开始时依次选择**行业**（Industry）和**领域**（Domain）。行业决定市场规模和竞争格局，领域决定开源 vs 商业化的天然倾向与专属事件。

---

## 行业（Industry）

### I-1. 云基础设施与 DevOps

**市场特征**：技术采购者为工程师，开源生态高度成熟

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 大（$500B TAM） |
| 竞争强度 | 极高（AWS / Azure / GCP 三巨头） |
| 开源友好度 | 极高 |
| 商业化难度 | 中（PLG 路径成熟） |

**特殊规则**
- 云厂商竞争事件出现频率 +30%
- 开源策略卡「Kubernetes 插件生态」可用
- 被云厂商集成 = 流量红利，但议价权丧失（商业化指数上限 -15%）

---

### I-2. 企业 SaaS / B2B 软件

**市场特征**：销售主导，合同周期长，CLV 高

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 极大（$650B TAM） |
| 竞争强度 | 高 |
| 开源友好度 | 低（买家不关心是否开源） |
| 商业化难度 | 低（价值主张清晰） |

**特殊规则**
- 企业销售策略卡效果 +20%
- 每 4 回合触发一次「年度续约谈判」事件
- 开源策略卡效果 -10%（企业买家不认这张牌）
- 可解锁「Gartner 魔力象限」特殊事件

---

### I-3. AI / 机器学习平台

**市场特征**：模型开源是惯例，但推理/训练服务是金矿

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 爆发期（增速 > 50% YoY） |
| 竞争强度 | 极高且快速变化 |
| 开源友好度 | 极高（社区即护城河） |
| 商业化难度 | 中高（monetize 模型难） |

**特殊规则**
- 每 3 回合随机出现「大模型发布震撼市场」事件
- 开源模型权重 = 开源指数大幅提升机会
- 可解锁「模型即服务（MaaS）」商业化路径（专属策略卡）
- GPU 成本作为隐性负担：现金储备每回合自动 -2%（AI 训练耗钱）

---

### I-4. 网络安全

**市场特征**：信任至关重要，开源透明度是双刃剑

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 大（$200B TAM） |
| 竞争强度 | 中高 |
| 开源友好度 | 中（开源扫描器常见，但核心引擎闭源） |
| 商业化难度 | 低高两极（工具免费，企业版昂贵） |

**特殊规则**
- 安全漏洞事件出现时，若为开源项目，公众曝光风险 +50%
- 若为闭源项目，漏洞修复速度 -30%（社区无法帮忙）
- 企业合规要求事件频率 +20%
- 政府/军事客户事件专属（要求闭源）

---

### I-5. 开发者工具与 IDE

**市场特征**：工程师即用户即采购者，口碑驱动

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 中（$25B TAM） |
| 竞争强度 | 高（VSCode 主导） |
| 开源友好度 | 极高 |
| 商业化难度 | 高（开发者付费意愿低） |

**特殊规则**
- 社区影响力对月度营收的转化率最低（开发者不爱付钱）
- 但社区影响力 > 70% 时，可触发「大厂收购邀约」结局
- 竞品 Fork 事件出现频率最高（你的工具很容易被复制）

---

### I-6. 金融科技 / FinTech

**市场特征**：监管严苛，信任成本高，闭源为主

| 属性 | 初始值 |
|------|-------|
| 市场规模 | 极大（$300B TAM） |
| 竞争强度 | 中（受监管保护） |
| 开源友好度 | 极低（合规要求闭源） |
| 商业化难度 | 极低（高利润行业） |

**特殊规则**
- 开源策略卡受到「合规限制」：每打出一张，需额外支付「合规成本」（现金储备 -5%）
- 监管审查事件出现频率 +40%
- 商业化路线强制解锁「持牌经营」加成（月度营收上限 +50%）

---

## 领域（Domain）

在选定行业后，进一步选择具体领域，获得专属策略卡和初始状态修正。

### 数据库 / 数据基础设施

**代表公司**：MySQL, MongoDB, ClickHouse, DuckDB

- 开源指数初始 +15%
- 专属策略卡：「托管云服务」（将开源项目商业化为 DBaaS）
- 核心张力：云厂商白嫖你的开源数据库做托管服务

### API 网关 / 中间件

**代表公司**：Kong, Nginx, Envoy

- 技术债务初始 -10%
- 专属策略卡：「企业网关订阅」
- 核心张力：开源版本和企业版本的功能边界在哪里

### 容器 / 编排

**代表公司**：Docker, Rancher, k0s

- 社区影响力初始 +15%
- 专属事件：「CNCF 孵化申请」（开源路线里程碑）
- 核心张力：开源生态标准化后，商业化空间被压缩

### 大语言模型工具链

**代表公司**：LangChain, Ollama, vLLM

- 开源指数初始 +20%，商业化指数初始 0%
- 专属策略卡：「托管 API 服务」
- 核心张力：开源社区期望免费，但推理成本极高

### 低代码 / No-Code 平台

**代表公司**：Appsmith, Budibase, n8n

- 商业化指数初始 +10%
- 专属策略卡：「SaaS 云托管版」、「私有化部署企业版」
- 核心张力：自托管用户 vs SaaS 付费用户的分化

### 监控 / 可观测性

**代表公司**：Grafana, Prometheus, OpenTelemetry

- 社区影响力初始 +10%，品牌声誉 +10%
- 专属策略卡：「云托管监控 SaaS」
- 核心张力：Prometheus 免费了，Grafana Cloud 怎么收费

## 行业 × 领域推荐搭配

| 行业 | 领域 | 游戏风格 |
|------|------|---------|
| 云基础设施 | 容器/编排 | 开源生态战（CNCF 路线） |
| 云基础设施 | 数据库 | 对抗云厂商白嫖 |
| AI 平台 | 大语言模型工具链 | 模型开源 + 推理商业化 |
| 企业 SaaS | 低代码平台 | 双轨经典玩法 |
| 开发者工具 | API 网关 | 开源 → 收购退出 |
| 网络安全 | 监控/可观测性 | 开源透明度悖论 |
